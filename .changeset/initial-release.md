Initial release of @sylphx/webgpu - WebGPU bindings for Node.js and Bun

Features:
- Complete WebGPU API implementation via wgpu-rs
- Support for compute shaders and rendering pipelines
- Cross-platform support (macOS, Linux, Windows)
- Node.js 18+ and Bun 1.3+ compatibility
- Comprehensive TypeScript definitions
- Full test suite with 37 tests and 16,538 assertions
- Complete documentation site
